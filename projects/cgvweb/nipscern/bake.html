Bake scene_data.bin

Este utilitário roda uma vez no navegador. Ele carrega ../geometry_data/CaloGeometry.glb, ../default_xml/JiveXML_*.xml e o WASM parser em ../parser/pkg/, aplica os thresholds, assa (bake) os mesh/edges/ghosts/tracks em um único binário compacto e faz download de scene_data.bin.

Pré-requisitos antes de rodar: npm run fetch:geometry:source e npm run fetch:samples (popula os arquivos fonte fora do repositório). Depois do bake, coloque scene_data.bin na pasta nipscern/.
