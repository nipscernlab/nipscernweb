NIPS-CERN Projects

The following research software developed by NIPS-CERN is featured on this website and is available as open source.

SAPHO Processor Architecture Framework
Simple Architecture for Processor Hardware and Organisation — FPGA processor design platform used for teaching digital logic at UFJF.
View project
YANC Compiler Toolchain
Yet Another NIPS-CERN Compiler — CMM-to-assembly and assembly-to-machine-code compiler for the SAPHO architecture.
View project
POLARIS Cross-platform IDE
Lightweight IDE for the SAPHO ecosystem. Runs on Linux and Windows; includes YAWT wave tracer and PRISM RTL viewer.
View project
AURORA Legacy
Former Windows-only Electron IDE for SAPHO (v1.0–v4.27.10, 2021–2023). Archived. Succeeded by POLARIS.
View project
CGV ATLAS Calorimeter Viewer
Interactive 3D visualisation of the ATLAS calorimeter system (TileCal, LAr barrel, HEC endcaps) built with Three.js and Vite.
View project
